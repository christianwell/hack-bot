## Commands

- `?help` – shows commands  
- `?ping` – simple test command  
- `?coinflip` – flips a coin  
- `?zachquote` – pulls a random Zach Latta quote from [zapi](https://zapi-seven.vercel.app/docs)  
- `?flavortownproject` – shares the link to the Flavortown project  
- `?flavorinfo "id"` – get data about a user  
- `?flavorcookies "id"` – get a user's cookies  
- `?flavorstore "id"` – get info about a Flavortown store item
